# 主播详情页面
- 鼠标放在**粉丝牌**上，可以按粉丝牌等级降序显示粉丝团成员（包括牌子已经灭了的）
- 点击**动态**，可以显示主播历史动态，每20分钟更新一次。所以在最坏的情况下，动态存活超过25分钟就会被记录。
- **30日内流水**计算方式为直播流水+充电流水。
- 直播的公会数据时间是2025/9/20
- 鼠标放在签名上，可以显示历史签名
- 点击大航海图上的点，可以显示当时在舰成员。
  - 用户名字右侧显示当月消费金额，计算方式为当月的礼物+SC，再加上当月的大航海的金额。所以如果在上个月上了提督，这个月在提督到期之前，仍然会算上这个提督钱。
  - 勾上Diff开关，可以对比两段时间内大航海成员差异（只能往后对比）
  - 勾上Inspect，可以对大航海用户活跃度进行分析，活跃度不足的会被划掉。
    - 这个功能比较慢，可能要等20秒左右
    - 测试用例：主播**礼拜六Liu** 2025年11月的大航海列表。
    - 结果仅供参考.
