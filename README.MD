# BiliMonitor
~~视奸你喜欢的up主~~
# Features
 - 定时记录关注列表内的up主的粉丝数
 - 定时获取特别关注列表内up主的动态，转发到邮箱
 - 记录直播间内弹幕，礼物，营收
 - 当主播开始直播时发送邮件提醒
 - 记录主播大航海
 - 有一个后台web，可以在上面查看记录到的数据
